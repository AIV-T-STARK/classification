Hồi âm:
Ngoài Front Page của MS ra còn rất nhiều phần mềm thiết kế web chuyên nghiệp và rất đẹp mắt khác như: Macromedia Flash MX 2004, hay Macromedia Dreamweaver MX 2004,v.v.. Nhưng theo tôi thì bạn nên dùng Macromedia Dreamweaver MX 2004. Nó rất hay. Ngoài ra bạn cần tìm hiểu các ngôn ngữ lạp trình web khác để hỗ trợ như : ASP,HTML, VBScript, JavaScript, SQL sarver, PHP...Tài liệu về Macromedia Dreamweaver MX 2004 bạn có thể tìm thấy rất nhiều ở các nhà sách. Ngoài ra bạn nên ghé thăm trang web: http://www.w3schools.com/ Tại trang web này bạn sẽ được học tất cả các ngôn ngữ lập trình web đấy. Chúc bạn thành công.
