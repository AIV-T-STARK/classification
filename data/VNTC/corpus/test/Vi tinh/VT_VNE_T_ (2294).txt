Ví dụ thực hiện 3 hệ điều hành(Windows 98, Windows XP Pro, Red Hat Linux 9.0).2)Xin hỏi là có bao nhiêu cách Install hệ điều hành Linux? Ưu và khuyết điểm của việc làm này? Nếu như không gán như vậy thì khi Install thì trình cài đặt sẽ thực hiện việc này ra sao?
Võ Hoài Thư
Hồi âm:
De cai nhieu he dieu hanh len mot may tinh (trong do co Linux), ban tien hanh phan chia o cung: - Mot phan vung co dinh dang FAT32 de cai dat Windows. - Mot phan vung khong de dinh dang gi de cai dat Linux. Ban cho dia cai Linux vao, nhung khong su dung Disk Druid ma tu dinh dang cac phan vung cua Linux De Install Linux ban co the su dung dia cai (mua ngoai cua hang dia), hoac install truc tiep tren Internet (neu ban co ADSL thi hay su dung cach nay). Khong can thiet phai mount cac thu muc vao cac phan vung khac nhau. Hi vong cau tra loi cua minh giup ich cho ban. 
Nguyen Minh Duc  
Chào bạn.1) Cài đặt nhiều HDH trên cùng một máy: Chương trình đầu tiên bạn phải có là boot manager. Chương trình này sẽ được BIOS gọi khi khởi động, đưa ra menu cho bạn chọn khởi động hệ điều hành nào nếu không sẽ thực hiện khởi động hệ điều hành default. Tuy nhiên Red Hat có sẵn boot loader, bạn chỉ cần cài Windows 98 trước, sau đó cài Windows XP ở một partition khác, rùi dùng CD Red Hat để install vào, lúc đó RedHat sẽ cho bạn chọn muốn boot partion nào trước. 2) Cách Install hệ điều hành Linux: Các bộ đóng gói Linux đã hỗ trợ rất tốt việc cài đặt, bạn chỉ cần dùng CD đóng gói đó boot sau đó chọn chế độ install bằng graphic hay text thôi, cả hai cách đều như nhau, nếu bạn chưa có kinh nghiệm thì có thể chọn option default sau đó dùng help để customize lại theo ý muốn. 3) Swap partition có vai trò gì trong hệ điều hành Linux: Nó giống như swap file của windows. Một số chương trình như oracle, DB2 có yêu cần riêng về swap space như tối thiểu 1200MB chẳng hạn . Nếu không có Swap partition thì hệ điều hành không hoạt động được.4) Cần thiết gán (mount) từng thư mục (vd: /mnt, /home, /var, /usr/bin/, ..) vào mỗi partition khác nhau. Việc này sẽ tối ưu hoá thao tác read/write trên các thư mục này. Nó giống như bạn cài đặt chương trình ở một partion khác  data được sử dụng để trên một partion khác. Nhưng nếu bạn mới cài đặt linux thì nên dùng chế độ default là đủ, khi cần cài đặt mốt phần mềm khác thì nó sẽ yêu cầu rieng được ghi ở document.
