Dong Ngo   
Cách ghi đĩa mp3 tương thích máy nghe nhạc Kenwood: Để các máy chơi nhạc mp3 (bao gồm cả máy nghe trên ô tô) chơi được mp3 ghi trên đĩa CD thì cần 3 điều kiện sau: 
1. Đĩa phải ghi ở định dạng Data-CD 
2. Các file mp3 phải được đặt tên theo bảng mã ASC-II (8bit). Nếu đặt tên file 16bit Unicode thì đa số các máy chơi nhạc sẽ không đọc được. 
3. mp3 phải theo chuẩn mà máy chơi nhạc quy định. Đây là lý do phổ biến nhất gây ra lỗi âm thanh lẹt xẹt như bạn mô tả. 
Các máy chơi nhạc đời cũ chỉ cho phép mp3 là CBR 128kbps, nghĩa là dữ liệu của file mp3 là dòng không đổi 128 kilobit/giây. Hiện nay, rất nhiều các file mp3 trên máy tính là chuẩn VBR (dòng dữ liệu thay đổi tuỳ theo sự phức tạp của đoạn nhạc) hoặc được lưu với bit-rate cao hơn 128Kbps. 
Để khắc phục hiện tượng như bạn đã nêu, bạn có thể sử dụng chương trình MusicMatch JukeBox (hoặc bất kỳ chương trình nào cho phép chuyển đổi bitrate) để chuyển các file nhạc về dạng 128Kbps CBR trước khi burn ra CD. 
Về lâu dài, bạn có thể mua một đầu máy nghe nhạc đời mới thì vừa có thể chạy tốt các file mp3 theo mọi định dạng, bạn còn có thể chơi cả file wma và một số loại file khác. ---------------- 
Thuật ngữ: - VBR: Variable Bit Rate - CBR: Constant Bit Rate - Kbps: KiloBit Per Second - wma: windows media audio
